Видалення

При видаленні аккаунту видаляються всі ваші данні а саме:

  • Дані користувача
  • Дані оселі
  • Наповнення оселі
  • Фото оселі
  • Оцінки, відгуки та рейтинг оселі
  • Оцінки, відгуки та рейтинг користувача
  • Угоди користувача
  • Угоди оселі
Погоджуюсь на видалення аккаунту та розумію наслідки
