Покроково як створити
Discussio угоду

Крок 1 Створення угоди
Угоду створює власник оселі. Формування відбувається шляхом обрання орендаря зі списку Дискусій або Мешканців.
Крок 2 Пропозиція
Після формування угода надходить орендарю на розгляд.
Крок 3 Формування акту
Після підтвердження, вона з'являється в ухвалених угодах. Тепер власник може сформувати акт прийому-передачі оселі.
Крок 4 Мешканець
Після формування акту власник може додати вас до мешканців, тим самим надати доступ до інформації про оселю
