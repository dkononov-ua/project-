Додати

Наповнення

Ця функція надає користувачам доступ до попередньо розроблених шаблонів та універсальної форми, яка полегшує процес внесення інформації про речі їх стан, а також інструкцій щодо використання.

Додати наповнення можна після створення профілю оселі

Створити

Оселю

Список предметів задієються для формування акту прийому-передачі оселі. А також для розуміння власників речей які знаходяться в оселі.
